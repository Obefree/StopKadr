# Где StopKadr хранит файлы на iPhone

**Рабочая версия для iPhone:** Expo Go или собранное приложение (одинаковая схема хранения).

## 1. Внутри приложения (автоматически)

Все проекты и кадры лежат в **песочнице приложения** — это память только для StopKadr. Путь на устройстве (служебный):

```text
…/Documents/StopKadr/Projects/<id-проекта>/
  project.json          ← название, FPS, настройки камеры, список кадров
  frames/
    000001.jpg
    000002.jpg
    …
```

Дополнительно:

```text
…/Documents/StopKadr/sync-settings.json   ← IP ПК для Wi‑Fi
```

**Когда сохраняется:** после каждого кадра, изменения настроек, удаления кадра.  
**После перезапуска** приложения или телефона проекты **остаются** (пока не удалите приложение / не «Стереть данные»).

В **«Файлах» iOS** эта папка **не видна напрямую** — так устроена iOS для Expo Go и обычных приложений.

## 2. Сохранить наружу (вы сами)

| Способ | Куда попадает |
|--------|----------------|
| **Сохранить** → **Архив (ZIP)** | «Файлы» iPhone, iCloud Drive, AirDrop на Mac/другой iPhone |
| **Сохранить** → **На ПК** | `Документы\StopKadr-Inbox\` на компьютере (Wi‑Fi) |
| **Export** (MP4) | «Фото» / галерея или «Файлы» (в сборке с FFmpeg, не в Expo Go) |

## 3. Expo Go vs своё приложение

| | Expo Go | StopKadr APK/IPA |
|--|---------|------------------|
| Проекты и кадры | В песочнице **Expo Go** | В песочнице **StopKadr** |
| Перенос | ZIP / ПК | ZIP / ПК |
| Удаление данных | Удалить Expo Go | Удалить StopKadr |

Смена Expo Go → будущий IPA: сделайте **ZIP** или **отправку на ПК** до переустановки.

## 4. Резервная копия (рекомендация)

Периодически: **Сохранить** → ZIP → «Сохранить в Файлы» → папка iCloud / «StopKadr backup».
